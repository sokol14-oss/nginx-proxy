# Тестовое задание: Цепочка Nginx Proxy и защита X-Forwarded-For

### Как запустить
```bash
docker-compose up --build
```

### Протокол тестирования (curl)

**1. Обычный запрос:**
```bash
curl http://localhost:8080
# Результат: Получен заголовок X-Forwarded-For: 172.20.0.1, 172.20.0.5, 172.20.0.4
```

**2. Попытка подмены IP (недобросовестный пользователь):**
```bash
curl -H "X-Forwarded-For: 1.1.1.1" http://localhost:8080
# Результат: Получен заголовок X-Forwarded-For: 172.20.0.1, 172.20.0.5, 172.20.0.4
```
*Вывод: Фейковый адрес `1.1.1.1` успешно отсечен на первом Nginx.*
